## Frameworks Clave de Lenny

### 1. Los 7 Poderes del Growth

1. **Viral loops** - Uso del producto trae más usuarios
2. **Content loops** - Content SEO/social que escala
3. **Paid marketing** - Ads rentables
4. **Sales** - Equipo de ventas
5. **Partnerships** - Distribución via partners
6. **Community** - Word of mouth orgánico
7. **Product-led growth** - El producto se vende solo

**Regla:** Las mejores empresas dominan 2-3 canales, no todos.

### 4. El Framework "Jobs to be Done"

**Pregunta:** ¿Para qué trabajo te "contrata" el usuario?

**Template:**
> "When [situación], I want to [motivación], so I can [resultado esperado]."

**Ejemplo Airbnb:**
> "When I'm planning a trip, I want to find unique local stays, so I can have authentic experiences."

### 5. Growth Loops vs. Funnels

**Funnel tradicional (lineal):**
```
Awareness → Interest → Desire → Action → (fin)
```

**Growth Loop (compuesto):**
```
Input → Action → Output → (reinicia como nuevo Input)
```

**Diferencia:** Los loops generan crecimiento compuesto, los funnels no.

## Insights de Airbnb (experiencia directa)

### El Referral Program
- **Mecánica:** Da $25, recibe $25
- **Clave:** Incentivo bilateral (giver + receiver)
- **Resultado:** 300% más efectivo que incentivo unilateral

### Photo Quality Hack
- Enviaron fotógrafos profesionales a listings
- **Resultado:** 2.5x más bookings
- **Lección:** A veces lo "unscalable" es lo más efectivo

### Trust Framework
- Reviews bidireccionales (host + guest)
- Verified ID
- Social connections (amigos de amigos)

## Aplicación a WhatsSound

### 1. Elegir los 2-3 Poderes de Growth

**Primarios (enfocar aquí):**
1. **Viral loops** - Audio compartido = distribución gratis
2. **Community** - Creators evangelizan

**Secundario:**
3. **Content loops** - Audio clips virales en TikTok/IG

**NO priorizar:**
- Paid marketing (ARPU muy bajo)
- Sales (B2C no requiere)

### 2. Diseñar el Growth Loop

```
┌─ Usuario crea audio ─────────────────────────┐
│                                               │
▼                                               │
Comparte en WhatsApp/IG ───────────────────────┤
│                                               │
▼                                               │
Receptor ve branded player ────────────────────┤
│                                               │
▼                                               │
CTA: "Responde con tu voz" ────────────────────┤
│                                               │
▼                                               │
Descarga app → Activa → Crea audio ────────────┘
```

### 3. Jobs to be Done de WhatsSound

**Job principal:**
> "When I want to communicate something emotional or personal, I want to send my actual voice, so I can express myself better than with text."

**Jobs secundarios:**
> "When I don't have time to type, I want to send a voice message, so I can communicate faster."

> "When I'm creating content, I want to add personality with audio, so I can connect deeper with my audience."

### 4. Referral Program Design

**Mecánica propuesta:**
- Invita a un amigo → Ambos reciben 1 mes premium gratis
- Share audio → Si receptor descarga, ambos ganan

**Incentivos:**
| Acción | Giver | Receiver |
|--------|-------|----------|
| Invita amigo | 1 mes premium | 1 mes premium |
| Amigo crea primer audio | 50 coins | 50 coins |
| Audio compartido llega a 100 plays | Badge especial | N/A |
