# OnlyFans
## Modelo de Contenido por Suscripción

### Perfil
- **Empresa:** Fenix International Limited
- **Fundación:** 2016, Londres
- **Fundador:** Tim Stokely
- **Stats (2024):** 4M+ creadores, 370M+ usuarios registrados
- **Revenue (2023):** $1.3B
- **Valoración:** ~$8B (2025)
- **Pagos a creadores:** $25B+ desde 2016

### Modelo de Monetización

#### Comisión de Plataforma
| Concepto | Split |
|----------|-------|
| **OnlyFans retiene** | 20% |
| **Creador recibe** | 80% |

**Modelo simple y transparente:** 80/20 en todo.

#### Fuentes de Ingreso para Creadores
1. **Suscripciones mensuales:** Precio fijo mensual
2. **Pay-per-view (PPV):** Contenido individual de pago
3. **Tips/Propinas:** Pagos voluntarios de fans
4. **Mensajes pagados:** DMs con contenido premium

### Estructura de Pagos

#### Pricing por Creadores
- **Suscripción mínima:** $4.99/mes
- **Suscripción máxima:** $49.99/mes
- **Tips:** Sin límite
- **PPV:** Precio libre

#### Earnings Promedio
- **Promedio anual creador:** ~$1,300/año (2023)
- **Top 1%:** Ganan la mayoría del dinero
- **Distribución:** Muy desigual (power law)

### Mínimos de Pago
- **Mínimo payout:** $20
- **Frecuencia:** Semanal disponible
- **Métodos:** Transferencia bancaria, Paxum
- **Tiempo procesamiento:** 3-5 días hábiles

### Modelo de Negocio Real
```
Fan paga suscripción $10/mes
├── OnlyFans: $2.00 (20%)
└── Creador: $8.00 (80%)

Fan envía tip $50
├── OnlyFans: $10.00 (20%)
└── Creador: $40.00 (80%)

Fan compra PPV $25
├── OnlyFans: $5.00 (20%)
└── Creador: $20.00 (80%)
```

### Por Qué Funciona el 80/20

#### Ventajas para Creadores
- **Simple:** Una sola comisión para todo
- **Generoso:** 80% es más que la mayoría de plataformas
- **Predecible:** Fácil calcular ganancias
- **Sin fees ocultos:** Todo incluido en el 20%

#### Para la Plataforma
- **20% de $25B+:** Negocio muy rentable
- **Escala:** Funciona con millones de creadores
- **Simplicidad operativa:** Un modelo para todos

### Lecciones para WhatsSound

#### Modelo 80/20 para Propinas
El más simple y atractivo para DJs:

```
Fan envía propina €10
├── WhatsSound: €2.00 (20%)
└── DJ recibe: €8.00 (80%)
```

#### Ventajas de Copiar OnlyFans
- **Fácil comunicar:** "Los DJs se quedan el 80%"
- **Competitivo:** Mejor que Twitch (50%), similar a YouTube
- **Atractivo para DJs:** Razón clara para usar la plataforma
- **Simple de implementar:** Sin tiers complicados

#### Aplicación a Suscripciones DJ
Si WhatsSound ofrece suscripciones:
- Misma comisión 80/20
- Consistencia en todo el producto
- Sin confusión para usuarios

### Propuesta para WhatsSound

| Feature | Implementación |
|---------|----------------|
| Split propinas | 80% DJ / 20% WhatsSound |
| Split suscripciones | 80% DJ / 20% WhatsSound |
| Mínimo payout | €20 |
| Frecuencia payout | Semanal |
| Payment processing | Incluido en el 20% |
| Comunicación | "Los DJs se quedan el 80%" |

### Consideraciones

#### ¿Por qué no 85/15 o 90/10?
- **80/20 es estándar reconocido**
- **Margen suficiente** para sostenibilidad
- **Permite mejoras futuras** (más servicios para DJs)
- OnlyFans lo validó con $25B en payouts

#### Alternativa: Modelo Tiered
```
Tier Básico: 80/20 (gratis para DJ)
Tier Pro: 85/15 (€9.99/mes)
Tier Premium: 90/10 (€29.99/mes)
```
