# Substack
## Suscripciones para Creadores de Contenido

### Perfil
- **Empresa:** Substack Inc.
- **Fundación:** 2017
- **Sede:** San Francisco
- **Modelo:** Newsletters de pago + contenido gratuito
- **Claim:** "Writers keep 90% of their revenue"

### Modelo de Monetización

#### Comisión de Plataforma
| Concepto | Porcentaje |
|----------|------------|
| **Substack** | 10% de ingresos |
| **Stripe (payment processing)** | ~2.9% + $0.30 |
| **Creador recibe** | ~87% neto |

#### Estructura Simple
- **Gratis empezar:** Sin costos iniciales
- **Solo pagas cuando ganas:** Modelo revenue share
- **Sin límites:** Publicar contenido gratis ilimitado

### Modelo de Suscripción

#### Para Lectores
- **Free tier:** Contenido gratuito, newsletters
- **Paid tier:** Contenido premium, acceso completo
- **Precio típico:** $5-15/mes (creador decide)

#### Para Creadores
- **Publicar gratis:** Sin restricciones
- **Activar pagos:** Cuando estés listo
- **Mix content:** Posts gratis + posts de pago
- **Email list:** 100% propiedad del creador

### Ejemplo de Earnings

```
Calculadora Substack:
- 1,000 suscriptores pagando $5/mes
- = $5,000/mes bruto
- - $500 Substack (10%)
- - ~$145 Stripe fees
- = ~$4,355/mes neto (~$52k/año)
```

### Por Qué Funciona 90/10

#### Comparación con Modelo Ads
```
Modelo Ads (típico blog):
- Necesitas 40,000 pageviews/día
- Para ganar $1,000/mes
- Dependes de algoritmos

Modelo Substack:
- 1,000 suscriptores a $5
- = $4,500/mes
- Relación directa con audiencia
```

#### Ventajas del 90/10
- **Atractivo para creadores:** El mejor split del mercado
- **Simple comunicar:** "Te quedas el 90%"
- **Alineación:** Substack gana cuando tú ganas

### Mínimos de Pago
- **Mínimo payout:** Según Stripe (~$100)
- **Frecuencia:** Mensual
- **Ownership:** Lista de emails siempre tuya

### Lecciones para WhatsSound

#### Newsletter/Updates de DJ
DJs podrían ofrecer suscripciones tipo Substack:
- **Updates semanales:** Próximos eventos, nuevos tracks
- **Contenido exclusivo:** Setlists, behind the scenes
- **Early access:** Tickets prioritarios

#### Modelo 90/10 para Suscripciones
Si WhatsSound ofrece suscripciones:

```
Fan paga €10/mes suscripción DJ
├── WhatsSound: €1.00 (10%)
├── Payment processing: ~€0.59
└── DJ recibe: ~€8.41 (84%)
```

#### Aplicación: "DJ Updates"
| Tier | Precio | Incluye |
|------|--------|---------|
| Free | €0 | Notificaciones de eventos |
| Fan | €5/mes | + Setlists, early access tickets |
| Superfan | €15/mes | + Exclusivos, requests, meet & greet virtual |

### Propuesta para WhatsSound

| Feature | Implementación |
|---------|----------------|
| Suscripciones DJ | Estilo newsletter |
| Comisión | 10% (como Substack) |
| Contenido free | Eventos, básicos |
| Contenido paid | Exclusivos, early access |
| Email list | DJ mantiene ownership |
| Exportable | Sí, lista de fans |

### Diferenciación Propinas vs Suscripciones

```
Propinas (estilo OnlyFans): 80/20
- Impulso del momento
- Variable
- Durante eventos

Suscripciones (estilo Substack): 90/10
- Recurrente
- Predecible
- Entre eventos
```
