# Patreon
## Modelo de Suscripciones para Creadores

### Perfil
- **Empresa:** Patreon Inc.
- **Fundación:** 2013
- **Sede:** San Francisco, CA
- **Modelo:** Suscripciones mensuales/anuales de fans a creadores

### Modelo de Monetización

#### Comisión de Plataforma
| Concepto | Porcentaje |
|----------|------------|
| **Comisión Patreon** | 10% de ingresos |
| **Payment processing** | ~2.9% + $0.30 |
| **Payout fees** | Variable según método |
| **Currency conversion** | ~2.5% si aplica |

**Total típico que pierde el creador:** ~13-15% de cada pago

#### Modelo de Suscripción
- **Suscripciones mensuales:** Recurrentes automáticas
- **Suscripciones anuales:** Descuento incluido
- **Tiers/Niveles:** Múltiples precios con diferentes beneficios
- **One-time payments:** Pagos únicos también disponibles

### Características de Monetización

#### Para Creadores
- Audio, imágenes, video nativo, livestreaming
- Newsletter por email
- Chats, DMs y comentarios con patrons
- Lista de emails exportable (ownership)
- Analytics y insights de miembros
- Herramientas de crecimiento automatizadas
- Descuentos, trials gratis, sales

#### Gestión de Pagos
- **Patreon maneja todo:** Sin necesidad de Stripe propio
- **Chargebacks/Disputes:** Patreon los gestiona
- **Fraude:** Protección incluida
- **Impuestos:** Patreon recauda y remite

#### Métodos de Pago Soportados
- Tarjetas de crédito/débito
- PayPal
- Apple Pay
- Venmo
- Sofort
- +16 divisas para payouts

### Mínimos y Payouts
- **Mínimo para empezar:** $0 (gratis crear cuenta)
- **Mínimo payout:** Varía por método (~$25-50)
- **Frecuencia payout:** Mensual (configurable)
- **Tiempo de pago:** 2-5 días hábiles

### Modelo de Negocio Real
```
Patron paga $10/mes
├── Patreon: $1.00 (10%)
├── Payment processing: ~$0.59 (2.9% + $0.30)
└── Creador recibe: ~$8.41 (84.1%)
```

### Lecciones para WhatsSound

#### Aplicación: Suscripciones a DJs
Fans podrían suscribirse a sus DJs favoritos:
- **Tier Básico ($3-5/mes):** Acceso a setlists, notificaciones de eventos
- **Tier Premium ($10-15/mes):** Contenido exclusivo, tracks unreleased
- **Tier VIP ($25+/mes):** Acceso backstage virtual, requests prioritarios

#### Ventajas del Modelo Patreon
- **Ingresos recurrentes:** Predecibles para DJs
- **Comunidad:** Engagement directo fan-DJ
- **Ownership:** DJs mantienen lista de fans
- **Churn bajo:** Suscriptores leales

#### Comisión Sugerida WhatsSound
- **10% base** (igual que Patreon)
- **Payment processing:** Pass-through al DJ o incluido
- **Justificación:** Mercado acepta 10% como estándar
